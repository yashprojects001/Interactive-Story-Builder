# 🎭⚡ Advanced Interactive Story Builder

🌌 *Not just a story… an experience.*
🚀 A fully interactive, game-like storytelling engine built using **HTML, CSS & JavaScript**

---

## 🌟 Project Highlights

✨ **Dynamic Story Engine**
→ Every choice changes the storyline

🎬 **Cinematic Typing Effect**
→ Story appears like a real game dialogue

🎒 **Inventory System**
→ Collect items that affect outcomes

❤️ **Health System**
→ Decisions impact survival

💾 **Save & Load Game**
→ Resume anytime using localStorage

🎵 **Background Music & Effects**
→ Immersive gameplay experience

🌙 **Dark / Light Mode Toggle**
→ Switch themes dynamically

📊 **Progress Tracking Bar**
→ Visualize your journey

✨ **Animated Particle Background**
→ Futuristic aesthetic

🏆 **Multiple Endings**
→ Hero | Legend | Survival | Game Over
